# Maryem Beauty Shop

**E-commerce platform for beauty and cosmetics — browse products, manage your cart, and shop seamlessly.**

</div>

---

## 🎯 Problem It Solves

Small beauty brands need a clean, fast online storefront that handles product browsing, cart management, and checkout — without the overhead of Shopify or similar platforms.

---

## ✨ Features

- Product catalog with category filtering
- Shopping cart with persistent state
- Supabase-powered backend (auth, database, storage)
- SEO-optimized with React Helmet
- Responsive design for mobile and desktop

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/Ah-med-hajji/maryem-beauty-shop.git
cd maryem-beauty-shop
npm install
```

### Environment Variables

```bash
cp .env.example .env
# Add your Supabase URL and anon key
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
